# Panda Sense

Panda Sense Environment Temperature Sensor

<img src={require('@site/docs/panda-series/panda-touch/panda-touch-extra/panda-sense/img/panda_sense_title.png').default} alt="" width="600" />

## **Specifications**

| Sensor Module            | AHT30                                   |
| -------------------- | --------------------------------------- |
| Connection Interface      | PH2.0 4-pin                             |
| Operating Voltage             | 3.3V                                    |
| Temperature Range & Accuracy   | -40℃ - 120℃, ±0.5℃ (10℃ - 60℃)       |
| Humidity Range & Accuracy   | 0-100%RH, ±3% (within 20%RH - 80%RH) |
| Product Dimensions             | 27.98 x 27.95 x 11mm                    |
| Cable Length             | 800mm                                   |
| Panda Touch Firmware Support | V1.0.6 and above                       |

## Installation Guide

**Prepare Tools**:

<img src={require('@site/docs/panda-series/panda-touch/panda-touch-extra/panda-sense/img/panda_sense_installation.png').default} alt="" width="600" />

**Installation Steps:**

:::tip

First magnetically attach the Panda Sense inside the machine, then route the connecting cable neatly within the printer.

The Panda Touch firmware must be version V1.0.6 or higher.

:::

1. Use a soldering iron to embed the hot melt nut into the printed part.

    <img src={require('@site/docs/panda-series/panda-touch/panda-touch-extra/panda-sense/img/panda_sense_installation1.png').default} alt="" width="600" />

2. Continue using the soldering iron to embed the two hot-melt nuts into the printed part, then secure the Panda Sense Dock to the printed part using M3x5 round-head screws.

    <img src={require('@site/docs/panda-series/panda-touch/panda-touch-extra/panda-sense/img/panda_sense_installation2.png').default} alt="" width="600" />

3. Secure the printed piece with the Panda Sense Dock attached to the Panda Touch USB Dock.

    <img src={require('@site/docs/panda-series/panda-touch/panda-touch-extra/panda-sense/img/panda_sense_installation3.png').default} alt="" width="600" />

    The final installation result is shown in the figure below:

    <img src={require('@site/docs/panda-series/panda-touch/panda-touch-extra/panda-sense/img/panda_sense_installation4.png').default} alt="" width="600" />

4. Install the Panda Sense Touch onto the Panda Touch I2C interface. The final installation should appear as shown in the image below:

    <img src={require('@site/docs/panda-series/panda-touch/panda-touch-extra/panda-sense/img/panda_sense_installation5.png').default} alt="" width="600" />

5. Insert the connection cable into the Panda Sense Dock's port. Once the Panda Touch magnetically attaches to the Panda Touch USB Dock, the Panda Sense will function normally.

    <img src={require('@site/docs/panda-series/panda-touch/panda-touch-extra/panda-sense/img/panda_sense_installation6.png').default} alt="" width="600" />
